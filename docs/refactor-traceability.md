# Refactor Traceability

Status after the architecture rewrite audit.

## Phase Coverage

| Phase | Status | Implementation Evidence |
| --- | --- | --- |
| 00 Baseline | Complete | Core/backend tests and shader build steps pass. |
| 01 Core API | Complete | `src/core/types.zig`, `src/core/units.zig`, root exports, `Transform`, `FrameToken`, `TextRun`. |
| 02 Font/Outline | Complete | `src/core/font/*`, `src/core/outline/{stream,regularize,area,encode}.zig`; native line/quad/cubic tests; `FontOptions.face_index` reaches FreeType loading. |
| 03 CoverageBlob | Complete | `src/core/blob/{format,encode,decode,hband,reference}.zig`; layout constants live in `format.zig`; command ABI uses backend-neutral `glyph_ref`; h-band candidates are tested against reference y-overlap and CPU point coverage. |
| 04 Render Core | Complete | `src/core/render/{renderer_core,glyph_store,text_batch,backend_contract}.zig`; `RendererCore` owns frame command generation while `GlyphStore` owns cache, pool, typed `GlyphRef` values, and deferred retirements; pointer and value backend shapes are compile-time checked. |
| 05 Shaders/ABI | Complete | `shaders/core/*`, `shaders/backend_{vulkan,metal}/resources.slang`, `shaders/entries/*`; ABI structs still generated by `tools/layout_gen.zig`. |
| 06 Vulkan | Complete | `src/backends/vulkan/*`; renderer exposes `Frame`, `Target`, `FrameToken`, `markFrameComplete`, and implements the backend contract. |
| 07 Metal | Complete | `src/backends/metal/*`; Zig sees C ABI handles only, borrowed host objects are documented and represented as non-retained ARC fields, and normal glyph retirement uses frame tokens instead of per-glyph full waits. |
| 08 Build | Complete | `build/{deps,c_libs,shaders,backends,demos}.zig`; lazy dependencies and ThinLTO policy preserved. |
| 09 Demo/Docs | Complete | `src/demo/common/*` shared scene; Vulkan/Metal demos use the same `Frame.drawText(TextRun)` path; README/AGENTS/CHANGELOG updated. |
| 10 Cleanup | Complete | Legacy roots removed, no source-level `@cImport`, no generated artifacts tracked, public root exports are explicit. |

## Pipeline Boundary Convergence

The follow-up plan in `docs/pipeline-boundary-refactor-plan.md` is implemented.

| Boundary | Status | Implementation Evidence |
| --- | --- | --- |
| FontSystem | Complete | `src/core/font/font_system.zig` owns FreeType library lifetime and returns `LoadedFont` values. |
| ShapePlan | Complete | `src/core/font/shape.zig` owns reusable HarfBuzz buffer state; `RendererCore` no longer manipulates `hb.Buffer` directly. |
| OutlineStream | Complete | `src/core/outline/encode.zig` captures HarfBuzz draw callbacks into `OutlineStream`. |
| RegularizedCubicSpans | Complete | `src/core/outline/regularize.zig` raises and splits native segments into `RegularizedCubicSpan` values. |
| CoverageBlob | Complete | `src/core/blob/format.zig` owns `CoverageBlob`; `src/core/blob/encode.zig` encodes regularized spans without `hb.Blob`. |
| GlyphStore | Complete | `src/core/render/glyph_store.zig` owns cache, byte pool, typed `GlyphRef` values, and deferred retirements. |
| TextBatch | Complete | `src/core/render/text_batch.zig` is a borrowed fixed-capacity writer used by Vulkan and Metal frames; `RendererCore.appendRun()` derives `Backend.Command` from the backend contract. |
| BackendFrame | Complete | Vulkan and Metal frames expose `drawText` and `submit`; backend contract uses `uploadBlob` and `retireBlob`. |
