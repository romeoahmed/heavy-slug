# Refactor Phase Plans

This directory expands `docs/architecture-refactor-spec.md` into executable phase plans.

The rewrite must keep the repository buildable at each checkpoint:

- Checkpoint A: phases 1-3
- Checkpoint B: phases 4-5
- Checkpoint C: phases 6-7
- Checkpoint D: phases 8-10

Each phase has acceptance criteria, verification commands, and likely file ownership.
